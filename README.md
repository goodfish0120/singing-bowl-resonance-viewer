# Singing Bowl Resonance Viewer

一個用於現場觀察頌缽聲音結構的即時視覺化小工具。

這不是音質評分器，也不是裂縫鑑定器。  
它只是把頌缽的聲音轉成一張比較直觀的共鳴圖，讓使用者用肉眼觀察主音、泛音、衰減與穩定度。

## 功能

- 使用手機麥克風即時分析聲音
- 每次偵測到新的敲擊聲，會自動重新開始顯示
- X 軸：敲擊後 0–30 秒
- Y 軸：相對音量，單位為 dB
- 可選顯示線數：4 / 8 / 12 條
- 顏色代表頻率相對主音的倍頻週期位置
- 不錄音、不儲存、不上傳資料

## 使用方式

1. 用手機瀏覽器打開網頁
2. 允許麥克風權限
3. 敲擊頌缽
4. 觀察圖上的共鳴線變化

每次重新敲擊時，畫面會自動重置，並重新抓取新的主頻。

## 圖像怎麼看

一般來說：

- 線條長、穩、平滑衰減：共鳴較乾淨
- 線條突然斷掉：可能有不連續的衰減
- 顏色快速漂移：頻率穩定度較差
- 短碎線或雜色很多：可能有 buzz、毛邊或環境雜訊
- 同一顆缽多次敲擊圖形差異很大：可能需要再確認敲擊方式或缽體狀態

這些只是視覺輔助，不代表自動判斷好壞。

## 建議使用環境

- 安靜一點的空間
- 手機與頌缽距離盡量固定
- 每次敲擊力道盡量接近
- 不要讓手機太靠近聲源，避免麥克風爆音

## 關於 HTTPS

手機瀏覽器通常需要在安全網址下才能使用麥克風。  
建議透過 GitHub Pages 開啟本工具。

直接從本機檔案打開 HTML 時，部分瀏覽器可能無法取得麥克風權限。
