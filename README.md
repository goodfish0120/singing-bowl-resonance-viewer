# Resonance Tuner

🌐 立即使用：
https://goodfish0120.github.io/singing-bowl-resonance-viewer/

Resonance Tuner 是一個用來觀察頌缽與共鳴樂器的網頁工具。

它會把一段聲音變成「共振指紋」，讓使用者用圖形看出這個聲音的結構：主頻在哪裡、有哪些泛音、哪些聲音比較強、哪些聲音在慢慢消失、整體形狀是否穩定。

工具直接在瀏覽器裡運行。
可以用麥克風即時分析，也可以選擇本機音檔分析。

## 這個工具想解決什麼問題

頌缽的聲音很複雜。

耳朵可以聽出「亮、厚、散、穩、乾淨、混濁」，但這些感覺很難用數字描述。

Resonance Tuner 嘗試把這些感覺轉成可以觀察的圖：

* 聲音裡有哪些主要頻率
* 哪些泛音比較強
* 泛音之間有沒有規律
* 聲音衰退時是否穩定
* 弱小的泛音尾端是否乾淨
* 不同頌缽之間的聲音形狀是否不同

## 基本使用方式

1. 打開網頁
2. 使用麥克風，或選擇一個音檔
3. 讓工具分析聲音
4. 調整 peak 敏感度
5. 觀察頻譜圖、泛音圓、呼吸圓
6. 換下一顆樂器時按「下一顆」

音檔分析目前會取前 15 秒。
這段時間通常包含最明顯的敲擊、泛音與衰退變化。

## 畫面在看什麼

### 1. 頻譜指紋

頻譜圖會顯示聲音中最明顯的頻率。

可以把它想像成聲音的山脈圖：

* 高的地方代表聲音比較強
* 尖峰代表明顯泛音
* 尖峰越清楚，代表那個頻率越突出

### 2. Peak 敏感度

不同樂器、不同錄音、不同手機麥克風，音量和細節都會不同。

敏感度滑桿用來校準要抓多少泛音：

* 敏感度低：只抓最明顯、最穩的泛音
* 敏感度高：會抓到更多細節與弱泛音

這個滑桿讓使用者可以依照實際聲音調整，而不是讓所有樂器套用同一個固定標準。

### 3. 泛音圓

泛音圓會把被選中的泛音變成圖形。

目前使用四種比例視角：

* log2：看倍頻與 octave 關係
* log3：看 3 倍比例關係
* log5：看 5 倍比例關係
* log7：看 7 倍比例關係

每一個圓都像是一個不同的鏡頭。
如果某個鏡頭裡的圖形比較整齊，代表這組聲音在那種比例關係下比較有結構。

### 4. 鏈式結構

泛音圓會把主要泛音依照音量由大到小接起來。

強泛音會形成主要骨架。
弱泛音會出現在尾端。

這樣可以更容易觀察：

* 聲音主體是否穩定
* 弱泛音是否乾淨
* 尾端是否出現毛刺
* 不同樂器的共振形狀是否不同

### 5. 呼吸圓

呼吸圓看的是聲音的能量變化。

頌缽敲下去後，聲音不會立刻消失。
它會慢慢變弱，有時還會出現一種像呼吸或拍動的變化。

呼吸圓用來觀察這些變化：

* 音量怎麼衰退
* 泛音能量怎麼起伏
* 聲音是否穩定收斂
* 是否有明顯的拍動感

## 怎麼判讀

圖形越整齊，通常代表這個聲音在該視角下越有結構。

可以用這樣的方式看：

* 頻譜尖峰清楚：主要泛音明顯
* 泛音圓整齊：泛音比例比較有規律
* 呼吸圓穩定：聲音衰退比較穩
* 尾端毛刺少：弱泛音比較乾淨
* 多次測試形狀相似：這個形狀更像樂器本身的特徵

這些圖形用來幫助觀察與比較。
最後仍然可以搭配實際聽感一起判斷。
