# Singing Bowl Resonance Viewer

🌐 **立即使用：**
https://goodfish0120.github.io/singing-bowl-resonance-viewer/

一個用手機麥克風即時觀察頌缽共鳴結構的視覺化工具。

它會在每次敲擊後自動擷取穩定頻率，將基準頻率與主要共鳴線轉成 0–30 秒的音量衰減曲線。
每條線的顏色代表該頻率相對基準頻率的位置，方便現場觀察共鳴是否穩定、尾音是否悠長、泛音是否清楚。


## 功能

* 即時使用手機麥克風分析聲音
* 每次偵測到新的敲擊聲，自動重新開始顯示
* X 軸：敲擊後 0–30 秒
* Y 軸：相對音量，單位為 dB
* 可選顯示線數：4 / 8 / 12 條
* 顯示主頻資訊
* 其他共鳴線以對應顏色標示頻率
* 不儲存、不上傳資料

## 使用方式

1. 用手機瀏覽器打開網頁
2. 允許麥克風權限
3. 敲擊頌缽
4. 觀察共鳴線的長度、穩定度與衰減形狀

## 圖像觀察重點

* 線條穩定、延伸較長：共鳴較持續
* 衰減平滑：聲音消失過程較自然
* 同一顆缽多次敲擊圖形接近：聲音反應較一致
* 短碎線較多：可能代表雜訊、氣音、敲擊干擾或不穩定成分
* 不同缽之間可用線條長度、穩定度與頻率分布做現場比較

## 建議使用環境

* 盡量選擇安靜環境
* 手機與頌缽距離保持一致
* 每次敲擊力道盡量接近
* 避免手機太靠近聲源造成爆音

## 技術說明

本工具使用瀏覽器 Web Audio API 進行即時音訊分析。
所有處理都在使用者裝置上完成，不需要後端服務。
