# Resonance Tuner

🌐 立即使用：
https://goodfish0120.github.io/singing-bowl-resonance-viewer/

Resonance Tuner 是一個瀏覽器端的共振指紋視覺化工具，適合用來觀察頌缽、金屬共鳴樂器，以及長延音聲源的頻率結構與動態形態。

工具可使用麥克風即時分析，也可讀取本機音檔。所有分析都在瀏覽器內完成，音檔不會上傳。

## 核心概念

Resonance Tuner 會從聲音中抽出一組共振特徵：

* 主頻
* 明顯泛音
* 泛音音量
* 音量差
* 拍動感
* 比例關係
* 動態軌跡

這些特徵會被整理成一組「聲音指紋」，用來觀察一顆樂器的共振形狀。

## 主要畫面

### 頻譜指紋

頻譜圖會顯示聲音中的主要頻率峰值。

使用者可以透過敏感度滑桿調整 peak 偵測強度，讓不同樂器、錄音音量、空間環境與麥克風條件都能被公平校準。

### 泛音圓

泛音圓會把被選中的主要泛音投影到四種比例空間：

* log2：倍頻 / octave 結構
* log3：3 系比例關係
* log5：5 系比例關係
* log7：7 系比例關係

每一組泛音會依照音量由大到小排列成鏈式結構。

強泛音決定主要骨架。
弱泛音出現在尾端。
尾端的抖動、分岔或毛刺可以幫助觀察聲音的細節穩定度。

### 呼吸圓

呼吸圓使用同樣的 log2 / log3 / log5 / log7 空間。

它關注音量差與能量起伏，用來觀察聲音敲擊後的呼吸感、拍動感與動態變化。

## 目前方向

Resonance Tuner 的目標是建立一個可運行的共振指紋模型。

基本流程：

1. 讀取聲音
2. 抽出共振特徵
3. 建立指紋模型
4. 讓指紋持續運行
5. 使用者切換樂器時清除並重新分析

這個專案正在測試哪些視覺結構能幫助使用者更直覺地比較真實共鳴樂器。
