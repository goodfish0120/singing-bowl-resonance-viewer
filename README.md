# Singing Bowl Resonance Tuner

🌐 **立即使用：**  
https://goodfish0120.github.io/singing-bowl-resonance-viewer/

一個用手機瀏覽器觀察頌缽基準頻率、即時頻譜與共鳴指紋的小工具。

第一屏採用進階調音器介面：大字顯示基準頻率、音名與 cents 偏差，搭配左右晃動的調音儀表與小型即時頻譜圖，方便現場拿著手機直接獲得視覺回饋。

往下滑可查看最近一次敲擊後 5–15 秒的頌缽頻率指紋，包括平均頻譜、上下範圍、主要泛音數量、泛音頻率、倍率關係、倍頻週期位置與延音長度。

## 主要功能

- 手機麥克風即時分析聲音
- 第一屏顯示調音器資訊與即時頻譜
- 偵測敲擊後自動建立本次聲音事件
- 使用敲擊後 5–15 秒作為穩定段，避開起音與尾端底噪
- 指紋圖顯示 median 頻譜與 P20 / P90 上下範圍
- 萃取主要共鳴峰數量與頻率
- 顯示各峰相對 F0 的倍率 ratio
- 顯示各峰在 octave cycle 裡的位置 `pos = frac(log2(f / F0))`
- 顯示 sustain 延音長度
- 所有分析都在裝置端即時完成

## 使用方式

1. 用手機瀏覽器打開 GitHub Pages 網址
2. 點擊畫面上的 `Tap to enable tuner`
3. 允許麥克風權限
4. 敲擊頌缽
5. 第一屏查看 F0、音名、cents meter 與即時頻譜
6. 敲擊後等待 15 秒左右，往下滑查看頻率指紋
7. 需要保存時，直接使用手機截圖

## 圖像觀察重點

第一屏適合快速確認：

- 目前基準頻率 F0
- 最接近音名與 cents 偏差
- 即時頻譜是否有明顯峰值
- 手機是否太靠近造成爆音

指紋頁適合比較：

- 主要泛音數量
- 泛音頻率分布
- 泛音相對 F0 的倍率
- 倍頻週期位置是否集中或分散
- 平均頻譜與上下範圍是否平滑
- sustain 是否足夠長

## 建議使用方式

- 使用同一支手機比較不同頌缽
- 手機與頌缽距離盡量固定
- 每次敲擊力道盡量接近
- 避免手機太靠近聲源造成爆音
- 在相似環境中比較不同樂器

## 技術摘要

- Web Audio API / `getUserMedia()`
- FFT 即時頻譜分析
- 敲擊事件偵測
- F0 估測與音名 / cents 轉換
- 5–15 秒穩定段取樣
- median + P20 / P90 頻譜指紋
- peak detection / ratio / octave position
